LearnGo 简介

LearnGo, a versatile learning tool with login and registration under GPT-4o-mini, enabling subject Q&A, smart note generation from uploaded materials, offline photo object recognition with translations, and multilingual voice input (Cantonese, English, Mandarin). It also features customizable themes for a personalized experience.

通过 RAG、NLP、CV、ML 等先进技术的结合，为用户提供智能、个性化的学习体验。
